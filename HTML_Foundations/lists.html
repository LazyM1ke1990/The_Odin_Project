Lists

Within HTML there are a couple types of lists at your disposal..

Unordered Lists:

Unordered lists are used if the order doesnt really matter, Like a shopping list.
This type can be created using the unordered list element '< ul >' and each idividual item
inside the list can be listed using the list item element '< li >'. The idividual list items in
an unordered list will begin with a bullet point.

  • item1
  • item2
  • item3

Ordered Lists:

Ordered lists are used if the order does matter, like in a step by step recipe. This list can
be produced using the ordered list element '< ol >' followed again by the individual items
in the list using the same '< li >' element. The difference is now the list will have a
numbered order (by default)

  1. item1
  2. item2
  3. item3

Bonus:

Lists can also be mixed and matched within each other.
Example:

  1. item1
  2. item2
    • item2a
    • item2b
  3. item3
