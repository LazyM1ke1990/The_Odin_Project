Working with Text

Paragraphs:

You can type text directly into the body of a html document but there would be no
formatting so you couldnt for example distinguish a paragraph from blocks of text. If paragraphs are made in the HTML code the browser would automatically condense
the spaces between the paragraphs into one block removing the spaces.

To setup a paragraph element you would use the '< p >' opening and closing tags.
Using these before and after a paragraph will help the browser know its a paragraph
your typing and format the text with spaces between each paragraph like it is supposed to.
This paragraph that is being read now has Paragraph tags around it, aswell as the above.

Headings:

There are six different levels of headings all being size and boldness. These headings
range from '< h1 > to < h6 >' h1 being the largest and most important. Its good practice
to only use one h1 element on each page.
Using the correct heading size is important to the pages structure as it provides a
hierarchy to the content.

Nesting and Indentation:

Indentation is used in HTML code when we nest elements inside of other elements.
This produces a 'Parent' and 'Child' relationship between them. The element that is nested
is known as the child element and the element it is nested inside of is known as the
parent.
Example would be the Paragraph element is nested inside of a body element.
The body element being the parent and the paragraph element being the child.
Parent elements can have multiple children elements at the same nesting level. for
example i could have two paragraph elements under the one body element.
To make these clear we use indentation on the child elements.

HTML Comments:

HTML comments are used to remind or annotate HTML code to help other users or
yourself in the future. These comments are not visible to the browser and are made by
enclosing comments in side '< !--' and ' -->' tags.
Example: < !-- This is a comment -- >
Shortcut: Ctrl + / comment selected line out automatically.
